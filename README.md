### **📂 Data Acquisition**
Due to the large file size of the full clinical dataset (approx. 28MB), the raw data is not hosted directly in this repository to maintain performance. To replicate this analysis, follow these steps:

1. **Visit:** [ClinicalTrials.gov](https://clinicaltrials.gov/)
2. **Search:** Enter `Breast Cancer` in the "Condition/Disease" field.
3. **Filter:** Select **"Completed"** or **"Active"** studies.
4. **Download:** - Click **"Download"** and select `CSV` format.
   - Ensure fields like *NCT Number, Title, Status, Conditions, Interventions, and Sponsor* are included.
5. **Setup:** Rename the file to `breast_cancer_trials.csv` and place it in your local project directory.
